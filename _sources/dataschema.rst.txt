Data Schema
===============

The purpose of this page is to document the ActivitySim data schema.
While the data schema can be different among implementations, the abm 
data tables (pandas DataFrames) and skims (numpy arrays) defined in the
:mod:`activitysim.abm.tables` package and used by the example, 
are described below.  The table definitions are a work-in-progress, and
only a partial set of fields is described, including 
table indexes, virtual (i.e. calculated on-the-fly) columns, initial input fields,
and model result fields.  The model that creates each field and if the field is 
virtual is also noted.

------------

skims from the omx_file on disk.  The injectables and omx_file for the example are listed below.

------------

The tours table contains tours attributes for each tour being simulated.  The tours 
table is created during the simulation as follows:

::

   tours.py -> non_mandatory_tours, mandatory_tours -> 
    util -> mandatory_tour_frequency.py, util -> non_mandatory_tour_frequency.py

Index - tour_id (see ``activitysim.abm.models.util.non_mandatory_tour_frequency.py``)

+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+

|                     person_id |           int64 |     util -> mandatory_tour_frequency.py, util -> non_mandatory_tour_frequency.py |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                     tour_type |          object |     util -> mandatory_tour_frequency.py, util -> non_mandatory_tour_frequency.py |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                      tour_num |           int64 |     util -> mandatory_tour_frequency.py, util -> non_mandatory_tour_frequency.py |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|   tour_departure_and_duration |     (start,end) |                             mandatory_scheduling.py, non_mandatory_scheduling.py |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                         start |           int64 |                                                                tours.py -> tours |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                           end |           int64 |                                                                tours.py -> tours |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                      duration |           int64 |                                                                tours.py -> tours |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                   destination |           int64 |                                                            destination_choice.py |          F |
+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+

|                          mode |           int64 |                                                                          mode.py |          F |

+-------------------------------+-----------------+----------------------------------------------------------------------------------+------------+
|                    out_period |           int64 |                                                                         tours.py |          T |
